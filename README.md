# Őńxx APK Tool
![install](https://files.catbox.moe/hspxgj.webp),
**Pro Level Python Tool** for converting `.apks` to signed APK and managing APK files.

## Features
- Direct `.apks` → Universal APK conversion (no temporary zip file)
- Automatic **V1 + V2 + V3** signing after conversion/move
- Move APK to any folder with auto-signing
- Beautiful colorful banner
- Opens your GitHub repo on startup
- Clean screen clearing
%|©## Requirements
- **Python 3.8+**
- `pip install colorama pyfiglet`
- `apksigner` in your PATH (from Android SDK Build-Tools)
- Java (if using advanced bundletool later)

## Installation
1. Download this repository or copy the code below.
2. Save as
 ```
 onxx_apk_tool.py
 ```
6. Install dependencies:
```
pip install colorama pyfiglet
```

```
git clone https://github.com/onxx-x143/APKS-APK.git
cd APKS-APK
chmod +x main.sh
```
